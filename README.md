# Site E-Commerce Vue.js

![Vue.js](https://img.shields.io/badge/Vue-3.2.47-brightgreen) ![Pinia](https://img.shields.io/badge/Pinia-State%20Management-blue) ![Firebase](https://img.shields.io/badge/Firebase-Auth-yellow)

## Description

Ce projet est un **site e-commerce moderne** développé avec **Vue 3** et **Vite**, utilisant :

- **API REST** pour la gestion des produits  
- **Pinia** pour la gestion du panier  
- **Firebase Authentication** pour l’authentification des utilisateurs  
- **HTML5 / CSS3 moderne** pour un design épuré et responsive  

Il permet de :

- Consulter les produits disponibles  
- Ajouter/supprimer des produits dans le panier  
- Gérer les quantités  
- Se connecter / se déconnecter via Firebase  

---

## Fonctionnalités

- Affichage des produits depuis une API externe  
- Gestion dynamique du panier avec Pinia  
- Calcul du total du panier en temps réel  
- Authentification sécurisée avec Firebase  
- Design responsive moderne avec animations simples  
- Navigation SPA via Vue Router
